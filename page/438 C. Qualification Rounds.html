#438 C. Qualification Rounds

time limit per test: 2 seconds

Snark and Philip are preparing the problemset for the upcoming pre-qualification round for semi-quarter-finals. They have a bank of problems, and they want to select any non-empty subset of it as a problemset.

experienced teams are participating in the contest. Some of these teams already know some of the problems. To make the contest interesting for them, each of the teams should know at most half of the selected problems.

Determine if Snark and Philip can make an interesting problemset!

Input

The first line contains two integers — the number of problems and the number of experienced teams.

Each of the next lines contains integers, each equal to 0 or 1. The -th number in the -th line is 1 if -th team knows -th problem and 0 otherwise.

Output

Print "YES" (quotes for clarity), if it is possible to make an interesting problemset, and "NO" otherwise.

You can print each character either upper- or lowercase ("YeS" and "yes" are valid when the answer is "YES").

Sample input

 
xxxxxxxxxx
5 3
1 0 1
1 1 0
1 0 0
1 0 0
1 0 0

Sample output

 
xxxxxxxxxx
NO

Sample input

 
xxxxxxxxxx
3 2
1 0
1 1
0 1

Sample output

 
xxxxxxxxxx
YES

Note

In the first example you can't make any interesting problemset, because the first team knows all problems.

In the second example you can choose the first and the third problems.

Answer

由于最大为4，每道题的状态最多只有16种，检查是否能选出两道题满足条件，如果不存在两道题满足条件，则说明选出任意两道题至少有一队两题都做过，则选出三道题时，由于至少有一队做过两题，也一定不满足条件，选出四题时，在四题中选两题有6种方法，假设每队最多只做过两题，则无法满足任意两道题至少有一队做过，则每队做过的题一定是超过两题的，所以也不满足条件，由于题数时，，所以选出任意两题无法满足条件的话，选出更多题也一定无法满足条件。
